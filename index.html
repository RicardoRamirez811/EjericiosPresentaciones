Desarrollo Web

Ing. Johny Cordon

Introducción al curso

El concepto de la web ha evolucionado desde su concepción y ahora se refiere a una poderosa plataforma que soporta aplicaciones potentes para las organizaciones. Con una amplia gama de tecnologías disponibles, los profesionales deben tener conceptos sólidos que les permitan no solamente elegir las mejores tecnologías que necesiten sino poder diseñar sistemas que aprovechen los mejores conceptos y las mejores tendencias para ofrecer soluciones de vanguardia. Al hablar de desarrollo web se debe enfocar en 2 áreas de dominio: desarrollo frontend y desarrollo backend, las cuales agrupan tanto conceptos como tecnologías para desarrollo de software que evolucionan constantemente y que tienden a especializar a los profesionales en una de ellas.

Objetivo General

  • Que el estudiante comprenda los conceptos fundamentales que rigen el uso de la plataforma web para el desarrollo de software sobre ella misma.

  • Motivar al alumno para crear en él el interés en la utilización de herramientas de programación aplicado a las necesidades de la plataforma web.

Resultados Esperados

Al finalizar el curso, el estudiante sera capaz de:

  • Explicar conceptos necesarios para el desarollo web

  • Construir una solución que integre de forma exitosa componentes de interfaz de usuario con componentes de aplicación de servidor.

  • Integrar una base de datos como componente de las soluciones de aplicaciones web

  • Reconocer y anticipar los problemas más comunes de seguridad informática relacionados a las aplicaciones web
